# Pi_Pico_Projects

大家一起來玩 Raspberry Pi Pico吧！ 享受這小家伙帶給大家的樂趣。

這裡的範例大多是自己參考官網創作出來的，歡迎多多使用，如有引用到其它創作者的內容亦會提供出處。接下來會陸續上傳實驗結果，敬請期待。

以下電路是本實驗參考之電路接線示意圖，主要提供二組LED輸出，二組按鍵輸入，一組半可變電阻供類比信號輸及一個I2C介面128x128像素的灰階OLED(SSD1327驅動晶片)顯示器。
